# Chip & Maggie: Kibble Quest

Reh Dogg Games Plus presents a lightweight HTML5 Canvas arcade maze game inspired by classic kibble-chasing action.

## Play

Open `index.html` in a browser or upload this folder to GitHub Pages.

Keep `intro.mp4` and `chipandmaggie-cover.jpg` in the same folder as `index.html` for the intro and arcade menu presentation.

## Controls

- Arrow Keys or WASD: move
- Space: pause
- Mobile: use the on-screen directional buttons

## Optional Audio

Place these files in the same folder as `index.html`:

- `chomp.wav`
- `theme_song.mp3`
- `kibble_chomp.wav`
- `powerup.wav`
- `power_mode_loop.mp3`
- `enemy_chase.wav`
- `enemy_eaten.wav`
- `bonus.wav`
- `hit.wav`
- `game_over_stinger.wav`

If any file is missing, the game still runs silently.

## Features

- Play as Chip or Maggie
- Kibble pellets and dog treat power-ups
- Ticks, fleas, and parasites
- Chipmunk and rabbit bonus chases
- Score, lives, levels, pause, victory, and game over
- Mobile-friendly touch controls
